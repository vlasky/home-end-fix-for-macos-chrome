# Changelog

## 1.0 - 2026-02-06

- Initial release
- Home/End key navigation for text inputs, textareas, and contenteditable elements
- Shift+Home/End for line-level text selection
- Ctrl+Home/End for field-level cursor movement
- Ctrl+Shift+Home/End for field-level text selection
- Leave non-text-field Home/End behavior to native Chrome
- Visual line boundary detection for soft-wrapped textareas
- Click toolbar icon to toggle on/off
- Shadow DOM focus detection for nested editable controls
- Browser-native fallback for input types without selection APIs
- Preserve user enable/disable state across extension updates
- Better textarea mirror sizing for visual line detection accuracy
